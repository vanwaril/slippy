Introduction

Slippy is a JavaScript library written by Jordi Boggiano. It enables:

Clean and easy HTML presentations

Amazing navigation capabilities

Pretty eye-candy!

HTML Format

Slippy lets you specify a slide as:

<div class="slide" data-parent="5">
    <h1>HTML Format</h1>
    <p>Slippy lets you specify a slide as:</p>
    <pre class="brush: html">
    ...
</div>

Powerful Navigation

Use arrow-keys, the space-bar, or mouse clicks to navigate normally.

Use the up arrow to go up one level from any slide.

Use [n] and [p] keys to move to the next or previous slide at the same level, skipping all sub-slides!

Navigate directly to a slide by typing in the number

Use [Esc] to go to a "preview" mode and chose your slide

Eye-Candy!

Use the provided JavaScript libraries to provide syntax-highlighting, custom layouts, and more!

Navigation

From a slide, you can move to the next-slide using [right]/[down] arrows or the [Page-Down] key

Press [right] to continue...

To skip this topic, press [n]

To go back to the previous topic (Introduction), press [p]

Overviews & Jumping

Overview: [ESC]/[Del]/[Tab] to view, [Click] to pick one slide

Jump: Press [0-9] keys followed by [Enter] to go straight to one slide

Please proceed to the next slide...

Parent Slides

The parent slide of this and the previous slide is the 'Navigation' slide.

Its parent slide is the first slide.

To navigate from any slide to its parent slide, press [up] arrow.

Layouts

This slide uses the "alt" layout

It inherits the class "nofooter" of the layout, which allows you to style it differently, and it doesn't have the footer div that the default layout has for example

By default, all slides use the default layout if it's present

Layouts must have a <content></content> tag that will be replaced by each slide's content

<div class="slide" layout="alt">
    Foo
</div>

<div class="layout" name="default">
    <h1>Some title</h1>
    <content></content>
    <div class="footer">
        Footer content
    </div>
</div>

<div class="layout nofooter" name="alt">
    <content></content>
</div>
